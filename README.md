# Art-Clin — Clínica Odontológica

Landing page premium para a **Art-Clin**, focada em alinhadores invisíveis (Smart Align), implantes e estética dental.

## Características
- Design Premium & Editorial
- Totalmente Responsivo (Desktop, Tablet, Mobile)
- Micro-animações fluidas (Intersection Observer)
- Slider de Antes/Depois interativo
- Integração direta com WhatsApp de agendamento
- Localização via Google Maps para todas as unidades

## Tecnologias
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Fraunces & Outfit)

## Como visualizar
1. Clone o repositório:
   ```bash
   git clone https://github.com/[seu-usuario]/art-clin.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador.
